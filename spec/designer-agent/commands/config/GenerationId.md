# GenerationId

## Назначение

Возвращает идентификатор генерации конфигурации.

## Синтаксис

```text
config generation-id [--extension=<extension name>]
```

## Параметры

- `--extension=<extension name>` — имя расширения.

## Связи

- Работает в группе `config`.

## Примечания

- Подтверждено встроенным `help config` в SSH shell `8.5.1.1150`.
