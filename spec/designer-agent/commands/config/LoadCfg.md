# LoadCfg

## Назначение

Загружает конфигурацию или расширение из файла.

## Синтаксис

```text
config load-cfg --file=<file> [--extension=<extension name>]
```

## Параметры

- `--file=<file>` — путь к входному файлу.
- `--extension=<extension name>` — имя расширения.

## Связи

- Парная команда: `config dump-cfg`.

## Примечания

- Подтверждено встроенным `help config` в SSH shell `8.5.1.1150`.
