# MobileClientDigiSign

## Назначение

Подписывает конфигурацию мобильного клиента приватным ключом.

## Синтаксис

```text
config mobile-client-digi-sign --file=<file>
```

## Параметры

- `--file=<file>` — путь к файлу.

## Связи

- Работает в группе `config`.

## Примечания

- Подтверждено встроенным `help config` в SSH shell `8.5.1.1150`.
