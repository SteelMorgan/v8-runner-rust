# MobileClientWriteFile

## Назначение

Сохраняет мобильный клиент в `.xml`-файл.

## Синтаксис

```text
config mobile-client-write-file --file=<file>
```

## Параметры

- `--file=<file>` — путь к результирующему файлу.

## Связи

- Работает в группе `config`.

## Примечания

- Подтверждено встроенным `help config` в SSH shell `8.5.1.1150`.
