# ManageCfgSupport

## Назначение

Изменяет параметры поддержки конфигурации.

## Синтаксис

```text
config manage-cfg-support [--disable-support] [--force]
```

## Параметры

- `--disable-support` — отключить поддержку конфигурации.
- `--force` — отключить поддержку даже при запрете изменения конфигурации.

## Связи

- Работает в группе `config`.

## Примечания

- Подтверждено встроенным `help config` в SSH shell `8.5.1.1150`.
