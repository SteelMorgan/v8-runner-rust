# DumpCfg

## Назначение

Сохраняет конфигурацию или расширение в файл.

## Синтаксис

```text
config dump-cfg --file=<file> [--extension=<extension name>]
```

## Параметры

- `--file=<file>` — путь к файлу результата.
- `--extension=<extension name>` — имя расширения.

## Связи

- Парная команда: `config load-cfg`.

## Примечания

- Подтверждено встроенным `help config` в SSH shell `8.5.1.1150`.
