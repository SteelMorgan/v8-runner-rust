# List

## Назначение

Выводит текущие значения shell-настроек агента.

## Синтаксис

```text
options list
```

## Параметры

- Команда не принимает параметров.

## Связи

- Показывает `notify-progress`, `notify-progress-interval`, `output-format`, `show-prompt`.
- Связана с `options get` и `options set`.

## Примечания

- Подтверждено runtime на `8.5.1.1150`.
