# ConnectIb

## Назначение

Подключает текущую SSH shell-сессию агента к информационной базе, указанной при старте `/AgentMode`.

## Синтаксис

```text
common connect-ib
```

## Параметры

- Команда не принимает параметров.

## Связи

- Используется перед командами, которым нужна активная сессия ИБ.
- Связана с `common disconnect-ib`.

## Примечания

- Подтверждено runtime на `8.5.1.1150`.
- В проверенном стенде завершалась `success` без дополнительного prompt к учетным данным ИБ внутри shell.
