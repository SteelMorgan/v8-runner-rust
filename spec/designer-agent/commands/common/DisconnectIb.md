# DisconnectIb

## Назначение

Отключает текущую SSH shell-сессию агента от ранее подключенной информационной базы.

## Синтаксис

```text
common disconnect-ib
```

## Параметры

- Команда не принимает параметров.

## Связи

- Завершает сеанс, открытый `common connect-ib`.
- Не завершает сам процесс агента.

## Примечания

- Подтверждено runtime на `8.5.1.1150`.
- В JSON-режиме возвращала `success`.
