# Shutdown

## Назначение

Завершает работу конфигуратора в режиме агента.

## Синтаксис

```text
common shutdown
```

## Параметры

- Команда не принимает параметров.

## Связи

- Завершает текущую SSH shell-сессию.
- Останавливает процесс агента.

## Примечания

- Подтверждено runtime на `8.5.1.1150`.
- После команды SSH-сессия закрывалась.
