# RestoreIb

## Назначение

Восстанавливает информационную базу из файла выгрузки.

## Синтаксис

```text
infobase-tools restore-ib --file=<file> [--threads=<n>]
```

## Параметры

- `--file=<file>` — файл выгрузки.
- `--threads=<n>` — число потоков.

## Связи

- Работает в группе `infobase-tools`.
- Связана с `dump-ib`.

## Примечания

- Подтверждено встроенным `help infobase-tools` в SSH shell `8.5.1.1150`.
- Разрушающее выполнение не проверялось в этом стенде.
