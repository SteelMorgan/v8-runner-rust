# EraseData

## Назначение

Удаляет данные информационной базы.

## Синтаксис

```text
infobase-tools erase-data
```

## Параметры

- Команда не принимает параметров.

## Связи

- Работает в группе `infobase-tools`.
- Является разрушающей операцией.

## Примечания

- Подтверждено встроенным `help infobase-tools` в SSH shell `8.5.1.1150`.
- В `help` явно указано, что команда доступна только пользователям с правом `Administration`.
