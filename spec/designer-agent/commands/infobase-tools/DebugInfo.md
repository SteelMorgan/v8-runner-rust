# DebugInfo

## Назначение

Возвращает настройки отладчика для текущей информационной базы.

## Синтаксис

```text
infobase-tools debug-info
```

## Параметры

- Команда не принимает параметров.

## Связи

- Работает в группе `infobase-tools`.
- Требует подключение к ИБ через `common connect-ib`.

## Примечания

- Подтверждено runtime на `8.5.1.1150`.
- В JSON-режиме возвращала `enabled`, `protocol`, `server-address`.
