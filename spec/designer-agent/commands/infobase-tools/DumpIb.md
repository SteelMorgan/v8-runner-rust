# DumpIb

## Назначение

Выгружает информационную базу в файл.

## Синтаксис

```text
infobase-tools dump-ib --file=<file>
```

## Параметры

- `--file=<file>` — путь к файлу выгрузки.

## Связи

- Парная команда: `infobase-tools restore-ib`.

## Примечания

- Подтверждено встроенным `help infobase-tools` в SSH shell `8.5.1.1150`.
