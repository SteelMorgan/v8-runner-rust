# AgentPort

## Назначение

Задает TCP-порт SSH-сервера агента.

## Синтаксис

```text
/AgentPort <Порт>
```

## Параметры

- `<Порт>` — номер TCP-порта для SSH-подключения к агенту.

## Связи

- Если параметр не указан, по умолчанию используется `1543`.
- В runtime подтверждены `1543` по умолчанию и явный `1544`.

## Примечания

- Подтверждено runtime и локальной help-страницей `zif3_AgentPort`.
