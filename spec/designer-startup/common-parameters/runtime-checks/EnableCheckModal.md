# EnableCheckModal

## Назначение

Включает строгую проверку использования модальных методов.

## Синтаксис

```text
/EnableCheckModal
```

## Связи

- В конфигурации с режимом модальности `Не использовать` или `Использовать с предупреждениями` вызов модального метода приведет к исключению или предупреждению.
