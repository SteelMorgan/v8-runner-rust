# IBName

## Назначение

Запуск информационной базы по имени базы в списке информационных баз.

## Синтаксис

```text
/IBName <имя информационной базы>
```

## Связи

- При необходимости имя должно быть взято в двойные кавычки.
- , если имя содержит двойные кавычки, то нужно заменить каждые двойные кавычки на две двойные кавычки.
- Если в списках баз будет найдено более чем одна база с указанным именем, то пользователю будет выдана ошибка.

## Примечания

- запуск информационной базы по имени базы в списке информационных баз.
