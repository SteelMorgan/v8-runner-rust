# SAOnRestart

## Назначение

Указывает на обязательность запроса пароля при перезапуске системы из данного сеанса работы (не используется в тонком клиенте).

## Синтаксис

```text
/SAOnRestart [+/-]
```

## Связи

- указывает на обязательность запроса пароля при перезапуске системы из данного сеанса работы (не используется в тонком клиенте).
- По умолчанию пароль не запрашивается.
