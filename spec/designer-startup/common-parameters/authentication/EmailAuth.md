# EmailAuth

## Назначение

Запускает аутентификацию через электронную почту.

## Синтаксис

```text
/EmailAuth <email>
```

## Связи

- Параметр не поддерживается в пакетном запуске конфигуратора.
- Параметр не работает в режимах, где запрещено отображение диалогов, включая `/DisableStartupDialogs`.

## Примечания

- На указанный адрес отправляется код подтверждения.
- После отправки кода клиент открывает окно для ввода подтверждения.
