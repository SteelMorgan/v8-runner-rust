# WSA

## Назначение

Определяет режим использования аутентификации операционной системы на веб-сервере.

## Синтаксис

```text
/WSA [+/-]
```

## Связи

- Если параметр `/WSA` не указывается, то подразумевается, что используется параметр командной строки `WSA+`.

## Примечания

- определяет режим использования аутентификации операционной системы на веб-сервере.
- `/WSA-` - запрет применения аутентификации пользователя на веб-сервере.
- `/WSA+` - установка применения аутентификации пользователя на веб-сервере.
- Используется аутентификация средствами операционной системы.
