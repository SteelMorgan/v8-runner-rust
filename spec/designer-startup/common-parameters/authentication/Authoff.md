# Authoff

## Назначение

Выполняет OpenID logout (завершение сеанса работы пользователя).

## Синтаксис

```text
/Authoff
```

## Примечания

- выполняет OpenID logout (завершение сеанса работы пользователя).
- Завершение сеанса работы выполняется вне зависимости от используемого в дальнейшем метода аутентификации.
