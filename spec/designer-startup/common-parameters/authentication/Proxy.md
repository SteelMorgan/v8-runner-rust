# Proxy

## Назначение

Явно задает настройки прокси и отключает использование значений по умолчанию.

## Синтаксис

```text
/Proxy -PSrv <адрес прокси> -PPort <порт> [-PUser <имя пользователя прокси> [-PPwd <пароль>]]
```

## Связи

- Параметр используется только в тонком клиенте.
- Параметр относится только к `ws`-соединению.
- `-PSrv` и `-PPort` обязательны.
- `-PPwd <пароль>` имеет смысл только вместе с `-PUser <имя пользователя прокси>`.

## Примечания

- Пример использования: `/Proxy -PSrv 192.168.0.10 -PPort 3128`.
