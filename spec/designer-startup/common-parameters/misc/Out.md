# Out

## Назначение

Задает файл для вывода служебных сообщений.

## Синтаксис

```text
/Out <имя файла> [-NoTruncate]
```

## Связи

- Если указан `-NoTruncate`, файл не очищается перед записью.
- В тонком клиенте `-NoTruncate` не используется.
- Во время выполнения пакетных команд файл сообщений можно открыть для просмотра.

## Примечания

- В пакетном режиме запись идет без буферизации.
- Файл формируется в кодировке `UTF-8`.
