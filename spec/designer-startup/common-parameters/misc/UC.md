# UC

## Назначение

Позволяет выполнить установку соединения с информационной базой, на которую установлена блокировка установки соединений.

## Синтаксис

```text
/UC <код доступа>
```

## Связи

- Если при установке блокировки задан непустой код доступа, то для установки соединения необходимо в параметре `/UC` указать этот код доступа.
- Не используется при работе тонкого клиента через веб-сервер

## Примечания

- позволяет выполнить установку соединения с информационной базой, на которую установлена блокировка установки соединений.
