# DisableStartupDialogs

## Назначение

Отключает стартовый диалог и диалоги аутентификации.

## Синтаксис

```text
/DisableStartupDialogs
```

## Связи

- Если командной строки недостаточно для выбора информационной базы, режима запуска или авторизации, вместо диалога команда завершится ошибкой.
- Если в командной строке ошибочная авторизация в хранилище конфигурации, будет ошибка.
- Если выполняется `CREATEINFOBASE`, а для кластера серверов задан администратор, будет ошибка вместо диалога.
- Если в командной строке нет авторизации в хранилище конфигурации, конфигуратор запускается без связи с хранилищем.

## Примечания

- В режиме `1С:Предприятие` параметр действует до окончания обработки события `ПередНачаломРаботыСистемы`.
- В режиме конфигуратора действие параметра заканчивается перед открытием основного окна.
- В пакетном режиме конфигуратора параметр действует весь сеанс, потому что основное окно не открывается.
