# DisableBackgroundIndexBuild

## Назначение

Отключает фоновое построение индекса рефакторинга при запуске конфигуратора.

## Синтаксис

```text
/DisableBackgroundIndexBuild
```

## Связи

- Отключает фоновое построение индекса рефакторинга при запуске конфигуратора.

## Примечания

- Индекс используется в следующих операциях: - поиск ссылок на объект конфигурации, - удаление объекта конфигурации;
- переименование объекта конфигурации.
