# Debug

## Назначение

Указывает, что данное клиентское приложение будет запущено в режиме отладки.

## Синтаксис

```text
/Debug [<режим>] [-attach]
```

## Связи

- Протокол, используемый для работы отладчика, определяется параметром <режим>: -tcp - для отладки используется протокол TCP`/IP`;
- Если в командной строке указан параметр -attach, то это означает, что отладчик будет автоматически подключать предметы отладки (клиентский и серверный) запускаемого приложения, которые будут зарегистрированы на сервере отладки.
- Параметр используется только для отладки по протоколу HTTP.

## Примечания

- http - для отладки используется протокол HTTP.
