# DebuggerURL

## Назначение

Идентификация отладчика, к которому приложение сразу после запуска должно подключиться.

## Синтаксис

```text
/DebuggerURL <URL отладчика>
```

## Связи

- Указывается URL отладчика (протокол, компьютер и номер порта), на котором в отладчике можно создавать удаленные объекты.

## Примечания
