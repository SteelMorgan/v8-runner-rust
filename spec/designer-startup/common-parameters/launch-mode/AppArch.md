# AppArch

## Назначение

Указывает разрядность используемого клиентского приложения на 64-разрядных Windows-системах.

## Синтаксис

```text
/AppArch <разрядность>
```

## Связи

- `<разрядность>` может принимать значения `x86`, `x86_64`, `x86_prt`, `x86_64_prt`.
- `x86` использует только 32-разрядные версии.
- `x86_64` использует только 64-разрядные версии.
- `x86_prt` ищет самую актуальную версию и при наличии обеих разрядностей выбирает 32-разрядную.
- `x86_64_prt` ищет самую актуальную версию и при наличии обеих разрядностей выбирает 64-разрядную.
