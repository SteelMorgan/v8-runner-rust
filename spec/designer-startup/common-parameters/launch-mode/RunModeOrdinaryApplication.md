# RunModeOrdinaryApplication

## Назначение

Запуск толстого клиента в обычном режиме, несмотря на настройки конфигурации и пользователя, от имени которого будет выполняться запуск.

## Синтаксис

```text
/RunModeOrdinaryApplication
```

## Связи

- Не используется в тонком клиенте.

## Примечания

- запуск толстого клиента в обычном режиме, несмотря на настройки конфигурации и пользователя, от имени которого будет выполняться запуск.
