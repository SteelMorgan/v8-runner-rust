# TestClient

## Назначение

Запуск толстого и тонкого клиента как управляемого другими клиентами с помощью менеджера тестирования.

## Синтаксис

```text
/TestClient [-TPort<Номер TCP-порта>]
```

## Связи

- TPort<Номер `TCP-`порта> - указывает номер порта для взаимодействия клиента и менеджера тестирования.
- По умолчанию используется порт 1538.
