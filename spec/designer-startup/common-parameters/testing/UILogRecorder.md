# UILogRecorder

## Назначение

Запись журнала интерактивных действий пользователя.

## Синтаксис

```text
/UILogRecorder [–TPort<Номер порта TCP/IP>] [-File<Путь>]
```

## Связи

- Журнал можно использовать для формирования сценария на встроенном языке "1С:Предприятие", позволяющего воспроизводить записанные действия.
- TPort<Номер `TCP-`порта> указывает номер порта для взаимодействия клиента и менеджера тестирования.
- Может совмещаться с параметром `/TESTCLIENT`.
- По умолчанию используется порт 1538.
- File<Путь> имя файла, в который будет сохраняться журнал действий пользователя после завершения записи, если к клиенту не подключён менеджер тестирования.

## Примечания

- запись журнала интерактивных действий пользователя.
