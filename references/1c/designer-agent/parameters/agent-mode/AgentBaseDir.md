# AgentBaseDir

## Назначение

Задает рабочий каталог для SFTP и для agent-команд загрузки/выгрузки конфигурации.

## Синтаксис

```text
/AgentBaseDir <рабочий каталог>
```

## Параметры

- `<рабочий каталог>` — базовый каталог файловых операций агента.

## Связи

- Локальная help-страница указывает default path для Linux: `~/.1cv8/1C/1cv8/<Уникальный идентификатор информационной базы>/sftp`.
- В runtime custom `AgentBaseDir` принимается на startup без ошибок.

## Примечания

- На стенде не удалось подтвердить runtime создание файлов внутри `AgentBaseDir`, потому что запросы прикладного уровня были заблокированы на этапе SSH userauth.
