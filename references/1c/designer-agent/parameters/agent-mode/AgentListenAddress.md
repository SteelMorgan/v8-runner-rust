# AgentListenAddress

## Назначение

Ограничивает IP-адрес, который прослушивает агент.

## Синтаксис

```text
/AgentListenAddress <Адрес>
```

## Параметры

- `<Адрес>` — IP-адрес для bind/listen.

## Связи

- Если параметр не указан, по умолчанию используется `127.0.0.1`.
- В runtime explicit и default запуск слушали `127.0.0.1`.

## Примечания

- Подтверждено runtime и локальной help-страницей `zif3_AgentListenAddress`.
