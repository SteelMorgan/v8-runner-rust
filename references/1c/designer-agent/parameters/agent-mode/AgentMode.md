# AgentMode

## Назначение

Включает режим агента конфигуратора.

## Синтаксис

```text
/AgentMode
```

## Параметры

Нет.

## Связи

- При наличии этого ключа игнорируются `/DisableStartupMessages` и `/DisableStartupDialogs`.
- В runtime на `8.3.27.1859` агент стартует headless и открывает SSH-порт.

## Примечания

- Подтверждено runtime и локальной help-страницей `zif3_AgentMode`.
