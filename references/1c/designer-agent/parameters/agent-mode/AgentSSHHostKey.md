# AgentSSHHostKey

## Назначение

Указывает путь к закрытому ключу хоста, который публикует SSH-сервер агента.

## Синтаксис

```text
/AgentSSHHostKey <приватный ключ>
```

## Параметры

- `<приватный ключ>` — путь к private key файла хоста.

## Связи

- Если `/AgentSSHHostKey` не указан, должен быть указан `/AgentSSHHostKeyAuto`.
- Если не указан ни один из этих параметров, запуск агента невозможен.

## Примечания

- На runtime-стенде явный ED25519 key был принят, и агент опубликовал соответствующий host key.
