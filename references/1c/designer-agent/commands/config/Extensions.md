# Extensions

## Назначение

Подрежим управления расширениями конфигурации.

## Синтаксис

```text
config extensions <subcommand> [options]
```

## Параметры

- `create` — создать расширение.
- `delete` — удалить расширение.
- `properties get` — получить свойства расширения.
- `properties set` — изменить свойства расширения.

## Связи

- Работает как submode внутри группы `config`.
- Связан с командами работы с расширениями конфигурации.

## Примечания

- Подтверждено встроенным `help config` в SSH shell `8.5.1.1150`.
