# DataSeparationCommonAttributesList

## Назначение

Выводит список разделителей информационной базы.

## Синтаксис

```text
infobase-tools data-separation-common-attributes-list
```

## Параметры

- Команда не принимает параметров.

## Связи

- Работает в группе `infobase-tools`.

## Примечания

- Подтверждено встроенным `help infobase-tools` в SSH shell `8.5.1.1150`.
