# Параметры Командной Строки Выбора Режима

## Назначение

Описывает верхнеуровневые формы запуска `1cv8` и выбор режима работы.

## Основные режимы

- `DESIGNER [<список параметров запуска>]` — запуск в режиме конфигуратора.
- `ENTERPRISE [<список параметров запуска>]` — запуск в режиме `1С:Предприятие`.
- `CREATEINFOBASE <строка соединения> [...]` — создание информационной базы.

## Структура Запуска

- Для `DESIGNER` список аргументов состоит из общих параметров запуска и, при необходимости, параметров пакетного режима.
- Для `ENTERPRISE` используются только общие параметры запуска.
- Для `CREATEINFOBASE` строка соединения задается в виде `ИмяПараметра=Значение`.

## Связи

- `CREATEINFOBASE` может использовать `/AddToList [<имя ИБ>]`, `/DisableStartupDialogs`, `/UseTemplate <имя файла шаблона>`, `/Out <имя файла>`, `/L`, `/VL`, `/O`, `/DumpResult <имя файла>`.
- Если `/AddToList` указан без имени, используется имя по умолчанию, аналогичное интерактивному созданию базы.
- `/UseTemplate <имя файла шаблона>` поддерживает шаблон из `cf` или `dt`.
- Если шаблон для `/UseTemplate` не указан, параметр игнорируется.
- `/DisableStartupDialogs` для `CREATEINFOBASE` отключает диалоги, которые иначе запрашивали бы данные у пользователя.
- Одновременное использование взаимоисключающих параметров выбора режима не допускается.

## Примечания

- Пример: `CREATEINFOBASE File=e:\\test3; /AddToList TEST33 /UseTemplate "C:\\Documents and Settings\\User\\My Documents\\tmplts\\TestVendor\\TestConfig\\1Cv8.cf"`.
