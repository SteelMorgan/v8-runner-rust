# OIDA

## Назначение

Включает или отключает OpenID-аутентификацию между информационными базами и внешними ресурсами.

## Синтаксис

```text
/OIDA[+/-]
```

## Связи

- Параметр применяется только в тонком и веб-клиенте.
- `/OIDA+` включает OpenID-аутентификацию и используется по умолчанию.
- `/OIDA-` отключает OpenID-аутентификацию.
- Если параметр не задан или задан `/OIDA+`, клиент пытается аутентифицироваться через OpenID-провайдера из `default.vrd`.
- Если OpenID-провайдер требует интерактивную аутентификацию, клиент открывает диалог ввода имени и пароля.
- Пользователь информационной базы должен совпадать по имени с пользователем базы OpenID-провайдера.

## Примечания

- Аутентификация идет по списку пользователей информационной базы OpenID-провайдера.
