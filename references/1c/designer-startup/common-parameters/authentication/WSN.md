# WSN

## Назначение

Имя пользователя операционной системы, которое будет использовано системой для выполнения аутентификации на веб-сервере в случае указания параметра `/WSA+`.

## Синтаксис

```text
/WSN <имя>
```

## Связи

- Имя пользователя операционной системы, которое будет использовано системой для выполнения аутентификации на веб-сервере в случае указания параметра `/WSA+`.
- Пароль пользователя указывается в параметре `/WSP`.
