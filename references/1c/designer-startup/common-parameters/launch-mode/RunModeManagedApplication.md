# RunModeManagedApplication

## Назначение

Запускает толстый клиент в режиме управляемого приложения.

## Синтаксис

```text
/RunModeManagedApplication
```

## Связи

- При указании параметра отключается механизм автоматического подбора клиентского приложения.

## Примечания

- Если в списке информационных баз указано `Выбирать автоматически`, запускается тонкий клиент.
- Если указано `Тонкий клиент`, запускается тонкий клиент.
- Если указано `Веб-клиент`, запускается веб-клиент.
- Если указано `Толстый клиент`, запускается толстый клиент в режиме управляемого приложения.
