# AppAutoCheckVersion

## Назначение

Управляет автоматическим подбором версии платформы для запуска клиентского приложения.

## Синтаксис

```text
/AppAutoCheckVersion [+/-]
```

## Связи

- `/AppAutoCheckVersion+` включает автоматический подбор версии платформы для каждой базы и используется по умолчанию.
- `/AppAutoCheckVersion-` отключает автоматический подбор версии платформы.
