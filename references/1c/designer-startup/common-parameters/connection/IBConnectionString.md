# IBConnectionString

## Назначение

Передает строку соединения с информационной базой целиком.

## Синтаксис

```text
/IBConnectionString <строка соединения>
```

## Связи

- `/IBConnectionString` должен стоять в командной строке раньше `/S` и `/F`, если они используются для переопределения частей строки соединения.
- Части строки соединения могут быть переопределены параметрами `/S` и `/F`.
- Саму строку соединения нужно брать в кавычки, а внутренние кавычки удваивать.
- В тонком клиенте строка соединения может включать параметры `wsn`, `wsp`, `wspauto`, `wspport`, `wspuser`, `wsppwd`, `wspsrv` для аутентификации и прокси на веб-сервере.

## Примечания

- Значение параметра соответствует формату, который возвращает функция `СтрокаСоединенияИнформационнойБазы()`.
