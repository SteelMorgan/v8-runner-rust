# SimulateServerCallDelay

## Назначение

Имитирует работу клиента при медленном соединении с сервером.

## Синтаксис

```text
/SimulateServerCallDelay [-CallXXXXX] [-SendYYYYY] [-ReceiveZZZZZ]
```

## Связи

- `-CallXXXXX` задает задержку вызова сервера в секундах; по умолчанию `4.45`.
- `-SendYYYYY` задает задержку отправки в секундах на каждый 1 Кбайт; по умолчанию `0.45`.
- `-ReceiveZZZZZ` задает задержку приема в секундах на каждый 1 Кбайт; по умолчанию `0.15`.
- Максимальное значение каждой задержки: `10` секунд.

## Примечания

- Пример использования: `/SimulateServerCallDelay -Call2.1 -Send1.3 -Receive1.2`.
