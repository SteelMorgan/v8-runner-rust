# HttpsForceTLS1_2

## Назначение

Принудительно использует TLS 1.2 или более раннюю версию при HTTPS-соединении.

## Синтаксис

```text
/HttpsForceTLS1_2
```

## Связи

- Параметр задает использование TLS 1.2 или более ранних версий при работе через веб-сервер по HTTPS.
- Нельзя одновременно указывать несколько параметров `/HttpsForceTLS*`.
- Если параметры `/HttpsForceTLS*` не указаны, клиент пытается использовать TLS начиная с версии 1.3, при необходимости понижая версию.
