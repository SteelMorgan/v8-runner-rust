# UseHwLicenses

## Назначение

Определяет режим поиска локального и сетевого аппаратного ключа защиты.

## Синтаксис

```text
/UseHwLicenses[+/-]
```

## Примечания

- `/UseHwLicenses+` - поиск аппаратного ключа защиты выполняется.
- `/UseHwLicenses-` - поиск аппаратного ключа защиты не выполняется.
