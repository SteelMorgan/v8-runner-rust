# TComp

## Назначение

Задает режим сжатия трафика между клиентом и сервером.

## Синтаксис

```text
/TComp [-None | -Deflate | -SDC | -lz4 | -zstd]
```

## Связи

- Параметр используется только в тонком клиенте.
- По умолчанию используется `-zstd`.

## Примечания

- `-None` отключает сжатие.
- `-Deflate` использует стандартное HTTP-сжатие `deflate`.
- `-SDC` использует собственный алгоритм сжатия платформы.
- `-lz4` использует алгоритм LZ4.
- `-zstd` использует алгоритм Zstandard.
