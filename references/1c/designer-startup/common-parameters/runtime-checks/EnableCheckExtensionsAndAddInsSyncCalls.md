# EnableCheckExtensionsAndAddInsSyncCalls

## Назначение

Включает режим строгой проверки использования синхронных вызовов расширений работы с файлами и криптографией и внешних компонент (используется для тонкого и веб-клиентов в режиме "1С:Предприятие").

## Синтаксис

```text
/EnableCheckExtensionsAndAddInsSyncCalls
```
